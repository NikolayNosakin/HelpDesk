Форма предназначена для выбора пользователя, группы пользователей, роли исполнителя или функции автоподстановки.

Выбор осуществляется кнопкой "Выбрать" (или двойным щелчком мыши) по соответствующему списку выбора. Если какой-либо из списков выбора отсутствует, то это означает что в текущем контексте работы выбирать значения данного вида не требуется, например, закладка функций автоподстановки доступна только при подборе из карточек шаблонов документов или процессов и недоступна при подборе из карточек самих документов или процессов.

На закладке пользователи можно выбрать вид отображения списка из ваиантов:

  • Списком - отображается один список пользователей.
  • По подразделениям - отображается иерерхический список подразделений и список пользователей.
  • По группам пользователей - отображается иерархический список групп пользователей и список пользователей. В этом режиме возможен выбор групп пользователей из списка.
