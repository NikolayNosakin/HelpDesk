Вариант отчета представляет собой сохраненную настройку, которая содержит описание группировок, отборов, условные оформления и т.п.

Для создания нового или изменения существующего варианта отчета можно воспользоваться командой "Сохранить вариант" из формы отчета.

Удаление отчета можно производить непосредственно из формы сохранения настроек варианта отчета, а также пометить вариант отчета на удаление непосредственно из списка вариантов отчета.

Для получения информация по настройке вариантов отчета обратитесь к документации по конфигурированию и администрированию 1С:Предприятие 8.
