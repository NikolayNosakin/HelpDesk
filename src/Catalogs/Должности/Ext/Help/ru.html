Справочник "Должности" предназначен для указания должностей в карточке сотрудника предприятия.

Для создания списка должностей в разделе "Нормативно-справочная информация" в панели навигации в группе "Предприятие" следует выполнить команду "Должности". В списке с помощью команд "Создать" и "Изменить" создавайте и редактируйте карточку должности.

С помощью команды "Пользователи" панели навигации карточки должности открывается список пользователей, имеющих данную должность.

С помощью команды "Присоединенные файлы" панели навигации карточки должности открывается список файлов, относящихся к данной должности (например, должностная инструкция).
