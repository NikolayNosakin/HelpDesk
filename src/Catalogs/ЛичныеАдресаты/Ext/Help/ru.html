Список Личных адресатов - это аналог персональной адресной книги. Текущий пользователь видит в списке только тех адресатов, которых добавил он сам.

При отправке исходящего письма в качестве получателя можно выбрать личного адресата. Также в программе есть возможность загрузить контакты из Microsoft Outlook (подменю Все действия-Загрузить из Microsoft Outlook)

Личный адресат может быть преобразован в корреспондента или контактное лицо.
