Как настроить кодировку

На закладке Письмо можно выбрать кодировку по умолчанию для новых писем, а также установить опцию автовыбора кодировки.

При установленной опции автовыбора в ответах и пересылках кодировка будет браться из письма-основания.

Как настроить подпись

Пользователь может сохранить несколько вариантов подписей к письмам, а также выбрать из них подпись для новых писем по умолчанию и подпись при ответах и пересылках по умолчанию. Подписи можно добавлять и редактировать в форме списка подписей пользователя, которая открывается по кнопке Подписи.

Для отключения автоматического включения в ответное письмо текста письма-основания необходимо снять флажок Вставлять текст исходного письма при ответе (по умолчанию установлен).
